Propriedades CSS

Propriedades Pra que serve?
width: ; É a proprieade que define a largura dos seus elementos.
height: ; É a propriedade que define a altura dos seus elementos.
margin: ; É a propriedade que define uma margem de distância entre outros elementos.
padding: ; É a propriedade que define um espaçamento dentro do elemento.
display: ; É a propriedade que define como o seu elemento vai ser renderizado no seu site.
font-size: ; É a propriedade que define o tamanho dos seus textos.
